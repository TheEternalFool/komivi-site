# Spec

## Project identity
A modern, text-first personal platform for Komivi Jarod Manuel Assiamua, combining portfolio, projects, writing, curated reading, and academic profile.

## Version 1 scope
- Home, About, Education, Projects, Writing, Reading, Chess, Contact
- Shared layout, responsive navigation, footer
- Reusable components
- Markdown/MDX-ready content architecture
- SEO metadata basics
- Accessible semantic HTML
- Mobile-first responsiveness

## Out of scope (v1)
- AI assistant, vector/semantic search, knowledge graph, CMS admin, auth, 3D, heavy animation

## Design direction
- Minimal, refined, modern, elegant, technical
- Text-first, calm hierarchy, dark-first aesthetic
- Restrained accent color, strong contrast
- Avoid generic startup aesthetics

## Content rules
- Do not invent personal facts
- Mark placeholders clearly
- Keep copy concise, thoughtful, professional
